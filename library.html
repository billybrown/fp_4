  • Free Press Actionkit Pattern Library

    This is the library of html, css and js patterns for the 'fp_4' Free Press Actionikit template sets. Actionkit templates are written in HTML with a spattering of Django. If you're looking to do something more complicated then the patterns here - I'd recommend learning Django and reading the Actionkit User Guide.

    Ideals

    1. Modular - All of these elements are meant to be combined and repeated to create the final design.
    2. Responsive - Everything should look good no matter what the screen size.
    3. Future Friendly - Everything is designed to be added to and modified.

    Rules

    1. Only Classes Add Styles - Styles will only be applied through classes, not IDs and not elements (with a few exceptions).
    2. Update This Guide - This style guide is meant to be updated. When you need to add new CSS, please add it to the SASS in the appriopriate partial. If you are in a rush or don't know where to put it, add it to _shame.scss and then properly incorporate it later.
